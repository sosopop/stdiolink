# 错误码参考

stdiolink 使用标准 HTTP 风格的错误码。

## 标准错误码

| 错误码 | 名称 | 说明 |
|--------|------|------|
| 0 | Success | 成功 |
| 400 | Bad Request | 参数错误 |
| 404 | Not Found | 命令不存在 |
| 500 | Internal Error | 内部错误 |

## 验证错误码

| 错误码 | 说明 |
|--------|------|
| 400 | 参数验证失败 |
| 401 | 缺少必填参数 |
| 402 | 类型不匹配 |
| 403 | 约束条件不满足 |
